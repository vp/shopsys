                    Toggle

                    Responsive toggle

                    Toggle Button
                    This is hidden div content
                    data-element: must point to ID of hidden/toggled element
                    data-hide-on-click-out: if you want to hide data-element on click on another place in document
                    When you click on button, it obtain state class "active"

                    Checkbox toggle

                    This is checbox show hide element content
                    data-checkbox-toggle-container-id: must point to ID of hidden/toggled element
                    data-checkbox-toggle-container-class: must point to class of hidden/toggled element
                    You can add "js-checkbox-toggle--inverted" class for inverted show/hide process. Do not remove original class "js-checkbox-toggle"
